# 基于 React, TypeScript 的旅游电商平台

**介绍**\
基于 React + TS 技术栈，利用 antd 组件库实现页面 UI 效果，Redux-Toolkit 实现全局状态管理模式，i18next 实现国际化语言切换功能，docker 进行容器化部署。\
\
